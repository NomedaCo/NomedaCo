**Nomeda** is a deep-tech AI lab building Arabic-first language models, tokenizers, and coding orchestration systems — openly on HuggingFace.

## Built in the Open

<br />

<div align="left">

> *"We build Arabic AI because it matters — and we build it in the open because that's how progress happens."*

<br />

- **Open Weights** — All Fattah-Orch model weights are publicly available on HuggingFace
- **Open Tokenizers** — Tokenizer configs and vocabularies freely accessible for both MSA and Egyptian Arabic
- **Open Datasets** — Fattah Orchestrator Dataset and Hindawi Arabic Sections released for community research
- **Research-First** — We prioritize linguistic accuracy and practical deployment over hype
